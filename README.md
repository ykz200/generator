## 项目仅供线动态切换连接数据库参考 
**项目说明** 

基于开源项目的代码生成器二次修改，可在线动态切换连接数据库生成bean,vo,service代码，根据项目需要减少开发工作量
本项目开源目的供代码参考springboot动态切换连接数据库

**本地部署**
- 通过git下载源码
- Eclipse、IDEA运行RenrenApplication.java，则可启动项目
- 项目访问路径：http://localhost：9898
